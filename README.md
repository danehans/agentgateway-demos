# agentgateway demos

Reproducible demos for evaluating agentgateway features in local Kubernetes
clusters.

## Demos

- [Cost-based semantic routing](cost-based-semantic-routing/): Route routine
  prompts to a lower-cost model and advanced prompts to a higher-capability
  model with agentgateway and vLLM Semantic Router, then measure cost, routing
  accuracy, satisfaction, and latency.
